| 离职类型             | 定义                                                         |
| :------------------- | :----------------------------------------------------------- |
| **辞职**             | 是指在任职期间内，有员工提出提前终止劳动关系的行为。         |
| **辞退**             | 是指在任职期间内，按照公司制度，员工符合辞退条件的，公司提前终止与员工的劳动关系，予以辞退的行为。 |
| **协商解除劳动关系** | 是指双方均可提出，经协商一致提前解除劳动关系，并签署解除协议。 |
| **自动离职**         | 是指在任职期间内，员工未经公司批准而擅自离开工作岗位的行为。 |
